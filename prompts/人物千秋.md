Build a single-page historical figure showcase web app called **人物千秋 (Renwu Qianqiu)** — an AI-powered cultural heritage platform where users enter any historical figure's name and get a museum-grade biographical page instantly.

Use **React 19 + Vite 8 + TypeScript + Tailwind CSS 3.4**. Add **Framer Motion** for animations and **lucide-react** for icons. Use **React Router** for routing (`/` homepage and `/figure/:id` detail page). Deploy on **EdgeOne Pages** with **Edge Functions (V8 runtime)** for backend APIs and **KV Storage** for data persistence.

The aesthetic is **dark museum style**: deep background, glass morphism cards, warm accent colors inspired by Chinese artifacts and historical pigments. The overall feel should be like walking into a museum exhibition hall.

## EDGE FUNCTIONS API

Create the following Edge Functions under `functions/api/`:

| Endpoint | File | Description |
|----------|------|-------------|
| `GET /api/figures` | `functions/api/figures.js` | List all figures (summary) |
| `GET /api/figures/:id` | `functions/api/figures/[id].js` | Single figure detail |
| `GET /api/random-figure` | `functions/api/figures/random.js` | Random figure |
| `GET /api/search?q=` | `functions/api/search.js` | Search figures by name |
| `GET /api/dynasties` | `functions/api/dynasties.js` | List dynasties |
| `GET /api/quotes` | `functions/api/quotes.js` | List famous quotes |
| `GET /api/stats` | `functions/api/stats.js` | Platform statistics |
| `GET /api/health` | `functions/api/health.js` | Health check |

Edge Function structure:
```javascript
export async function onRequest(context) {
  const { env } = context;
  // Access KV via env.KV_BINDING
  const data = await env.KV_BINDING.get('key', { type: 'json' });
  return new Response(JSON.stringify(data), {
    headers: { 'Content-Type': 'application/json' }
  });
}
```

KV Binding name: `KV_BINDING`

## ANIMATION GUIDELINES

- Use Framer Motion throughout
- Default easing: `[0.22, 1, 0.36, 1]` (restrained, elegant)
- Scroll-triggered animations: use `useInView` or Framer Motion `whileInView`
- Stagger children: use `staggerChildren` with `delayChildren: 0.1`
- Support `prefers-reduced-motion`

## DEPLOYMENT

1. Create Vite project: `npm create vite@latest -- --template react-ts`
2. Install deps: `npm install framer-motion lucide-react react-router-dom clsx tailwind-merge`
3. Configure EdgeOne Pages: create `edgeone.json` with `PAGES_SOURCE=skills`
4. Build: `npm run build`
5. Deploy: `edgeone pages deploy -n 'renwu-qianqiu'`
