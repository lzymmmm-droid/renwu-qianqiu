---

Build a single-page historical figure showcase web app called **人物千秋 (Renwu Qianqiu)** — an AI-powered cultural heritage platform where users enter any historical figure's name and get a museum-grade biographical page instantly.

Use **React 19 + Vite 8 + TypeScript + Tailwind CSS 3.4**. Add **Framer Motion** for animations and **lucide-react** for icons. Use **React Router** for routing (`/` homepage and `/figure/:id` detail page). Deploy on **EdgeOne Pages** with **Edge Functions (V8 runtime)** for backend APIs and **KV Storage** for data persistence. Load Google Fonts (Noto Serif SC for headings, Noto Sans SC for body).

The aesthetic is **dark museum style**: deep background, glass morphism cards, warm accent colors inspired by Chinese artifacts. The overall feel should be like walking into a museum exhibition hall.

═══════════════════════════════════════════════════
EDGE FUNCTIONS API
═══════════════════════════════════════════════════

Create 8 endpoints under `functions/api/`:

GET /api/figures — return all figures summary list:
```javascript
export async function onRequest(context) {
  const { env } = context;
  const figures = await env.KV_BINDING.get('figures', { type: 'json' });
  return new Response(JSON.stringify(figures), {
    headers: { 'Content-Type': 'application/json', 'Cache-Control': 'public, max-age=3600' }
  });
}
```

Endpoint list:
| Endpoint | File | Returns |
|----------|------|---------|
| GET /api/figures | functions/api/figures.js | All figure summaries |
| GET /api/figures/:id | functions/api/figures/[id].js | Single figure full data |
| GET /api/random-figure | functions/api/random-figure.js | Random figure |
| GET /api/search?q= | functions/api/search.js | Search results |
| GET /api/dynasties | functions/api/dynasties.js | Dynasty list |
| GET /api/quotes | functions/api/quotes.js | Famous quotes |
| GET /api/stats | functions/api/stats.js | Platform stats |
| GET /api/health | functions/api/health.js | { ok: true } |

KV Binding name: `KV_BINDING`

KV Data structure:
| Key | Type | Description |
|-----|------|-------------|
| figures | JSON | Array of figure summaries |
| figure:{id} | JSON | Single figure full data |
| quotes | JSON | Array of famous quotes |
| stats | JSON | Platform statistics |
| dynasty:{name} | JSON | Figures in a dynasty |

Figure data schema:
```typescript
interface Figure {
  id: string;
  name: string;
  courtesyName?: string;    // 字
  pseudonym?: string;       // 号
  birthYear?: number;
  deathYear?: number;
  dynasty: string;
  birthplace?: string;
  title: string;            // 称号
  summary: string;          // 一句话概括
  biography: string;        // 生平全文
  timeline: Array<{ year: string; title: string; description: string }>;
  achievements: Array<{ title: string; description: string; icon: string }>;
  works: Array<{ title: string; year?: string; description: string; type: string }>;
  quotes: Array<{ text: string; source: string }>;
  evaluations: Array<{ text: string; author: string; source: string; era: string }>;
  tags: string[];
  relatedFigures: string[];
  funFacts?: string[];
}
```
