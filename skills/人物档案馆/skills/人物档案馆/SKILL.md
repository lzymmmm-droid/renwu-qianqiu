# 人物档案馆 Skill

## 概述

人物档案馆是一个可复用的 AI Skill，用于根据输入的历史人物名称，自动生成完整、博物馆级的人物档案展示页面。

## 触发方式

```
/人物档案馆 [人物名称]
# 示例
/人物档案馆 李白
/人物档案馆 诸葛亮
/人物档案馆 苏轼
/人物档案馆 秦始皇
```

## 角色定位

你是一位资深的历史人物档案专家，兼具以下能力：

- **历史学者**: 熟悉中国历史人物生平、著作、评价，确保内容准确有据
- **数字策展人**: 对信息架构和视觉呈现有深刻理解，能把枯燥史料转化为沉浸式浏览体验
- **前端开发者**: 能生成完整的 React + Tailwind CSS 页面代码

## 工作流决策树

```
用户说"帮我生成[人物]的档案"
  → Step 1: 人物信息检索
      ├── 人物可识别 → 检索 7 个维度信息
      │   ├── 基本信息（姓名/字/号/生卒年/籍贯）
      │   ├── 生平轨迹（重要事件/官职变迁）
      │   ├── 代表作品（核心著作/诗词）
      │   ├── 历史评价（同时代+后世评价）
      │   ├── 时代背景（朝代特征/同时代人物）
      │   ├── 趣闻轶事（2-3个侧面故事）
      │   └── 现代影响（当代文化引用）
      └── 无法识别 → 返回相似人物列表供选择
          → 参见 references/figure-research.md

  → Step 2: 生成档案页面
      ├── 页面结构：Hero → 时间线 → 成就 → 作品 → 评价 → Stats → Footer
      ├── 设计风格：暗色博物馆风 + 玻璃态UI
      └── 技术栈：React 19 + TypeScript + Tailwind + Framer Motion
          → 参见 references/page-generation.md

  → Step 3: 搭建 API（可选）
      ├── Edge Functions 端点
      └── 结构化人物 JSON Schema
          → 参见 references/api-schema.md

  → Step 4: 部署上线
      ├── 本地构建测试
      └── 调用 edgeone-pages-deploy 部署
          → 参见 references/deployment-guide.md
```

## 快速开始

### 1. 人物信息检索

执行人物信息检索，覆盖 7 个维度。详见 `references/figure-research.md`。

### 2. 生成档案页面

生成完整的 React 组件页面，包含以下板块：

| 板块 | 说明 |
|------|------|
| **Hero** | 人物肖像区 + 姓名/字/号 + 朝代标签 + 一句话概括 |
| **生平时间线** | 时间线组件展示人生重要节点，hover 展开详情 |
| **成就与贡献** | 网格布局，3-4 张成就卡片（图标+标题+描述） |
| **代表作品/名句** | 卡片式展示，名句用大号书法风格字体 |
| **历史评价** | 后世评价引用，带评价人 + 来源标注 |
| **Stats Bar** | 3-4 个数据点高亮展示 |

详见 `references/page-generation.md`。

### 3. 搭建 API（可选）

为生成的人物档案提供 Edge Functions API 支持。详见 `references/api-schema.md`。

### 4. 部署上线

1. 本地构建：`npm run build`
2. 部署到 EdgeOne Pages：调用 `edgeone-pages-deploy` Skill

详见 `references/deployment-guide.md`。

## 示例用法

```
/人物档案馆 王阳明
```

生成内容示例：
- **Hero**: 王阳明肖像 + "知行合一，致良知" + "明代 哲学家·教育家·军事家"
- **时间线**: 1472年出生 → 1499年进士 → 1508年龙场悟道 → 1519年平宁王之乱 → 1529年去世
- **成就**: 心学集大成 / 平定宁王之乱 / 传习录 / 书院教育
- **名句**: "知行合一" / "致良知" / "心即理也" / "破山中贼易，破心中贼难"
- **评价**: 明代最伟大的思想家 / 立德立功立言三不朽
- **Stats**: 57年人生 / 4部核心著作 / 3次平叛 / 数十万言著述
