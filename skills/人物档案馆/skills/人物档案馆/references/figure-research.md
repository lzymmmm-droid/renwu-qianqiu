# 人物信息检索规范 (Step 1)

当用户调用「人物档案馆」Skill 时，第一步是根据人物名称检索以下 7 个维度的信息。

## 检索维度

| 维度 | 内容 | 优先级 |
|------|------|--------|
| 基本信息 | 姓名、字、号、生卒年、籍贯 | 必需 |
| 生平轨迹 | 主要人生阶段、重要事件、官职变迁 | 必需 |
| 代表作品 | 核心著作、诗词、理论贡献 | 必需 |
| 历史评价 | 同时代人评价、后世评价、争议点 | 必需 |
| 时代背景 | 所处朝代特征、同时代重要人物 | 推荐 |
| 趣闻轶事 | 2-3 个生动有趣的侧面故事 | 可选 |
| 现代影响 | 当代文化中的引用、纪念、研究 | 可选 |

## 人物识别规则

- **可识别**：名称在历史记载中有明确对应人物 → 正常检索
- **模糊匹配**：名称可能对应多个人物（如"韩信"有汉初韩信和韩王信）→ 返回列表供用户选择
- **无法识别**：名称无历史记载对应 → 返回可选的相似人物列表

## 信息质量标准

1. **准确性**：基于可靠历史记载，避免虚构或过度演绎
2. **有争议的评价**：注明不同观点，如"有史书记载...，但也有学者认为..."
3. **丰富度**：尽可能覆盖更多维度，使页面内容充实
4. **趣味性**：趣闻轶事应生动可读，能体现人物性格侧面
